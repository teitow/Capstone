TTS 보이스 세팅

1. 음성 명령을 통한 조정 ( 어렵진 않은데 조작이 난해할듯)
음성 인식 기능을 사용하여 톤과 속도를 조정
사용자가 "톤 높여", "톤 낮춰", "속도 빠르게", "속도 느리게"와 같은 명령 
Android의 SpeechRecognizer 클래스를 활용하여 음성 명령을 텍스트로 변환하고, 이를 통해 TTS 설정을 조정

2. 물리적 버튼 사용
볼륨 버튼이나 기타 물리적 버튼을 활용하여 TTS 설정을 조정
볼륨 업 버튼으로 톤이나 속도를 증가시키고, 볼륨 다운 버튼으로 감소시키는 방법
사용자가 버튼을 누를 때마다 TTS로 현재의 톤이나 속도를 음성으로 알려준다

3. 터치 기반 제스처 사용 ( 이게 좋을듯 기능 사용법이 이전과 일치하도록 )
특정 제스처를 통해 톤과 속도를 조절
화면을 좌우로 스와이프하여 속도를 조절하고, 상하로 스와이프하여 톤을 조절각 조정마다 TTS로 피드백을 제공하여 사용자가 현재 설정을 알 수 있게 함



그냥 사용법은 지우고, 매번 켤때마다 TTS로 사용법을 말해주자.
